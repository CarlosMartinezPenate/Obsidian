# Dino F/2 Scale-Up Protocol

## Goal
Scale all five F/2 dinoflagellate cultures from ~100 mL into larger fresh-medium volumes while preserving a backup line for each strain.
## References
- [[Xiang 2013]]: liquid maintenance in IMK at 24–26°C and low light; strain-specific responses, especially for SSA02 and SSE01.
- [[Symbiodiniaceae Culturing Protocols – Parkinson Lab]]

## Main strategy
- **F/2 only**
- Scale **all strains**
- Use **conservative inoculation**
- Preserve **one backup line per strain**
## Current starting point - 11/6/26
- each culture is ~100 mL
- vessel: **250 mL wide-mouth Erlenmeyers**
- closure: **gauze**
- cultures kept **static**
- incubator: **26°C**, **12:12**
- medium for scale-up: **fresh F/2 in ASW**

## Backup strategy
Since there are currently no true backups, the present ~100 mL cultures should be treated as the backup line.

### Backup plan
For each strain:
- keep the original ~100 mL culture as a **backup / maintenance line**
- remove only part of it as inoculum for the new scale-up culture
- refill the original culture with the same volume of **fresh F/2** that was removed
- create the new scaled culture in a **separate flask**

####  Backup refill
After removing inoculum for scale-up:
- refill the original backup culture back toward its working volume with **fresh F/2**
- mix gently
- return it to the incubator

## First scale-up step
### Recommended target
Scale each strain to **250 mL final volume** in a new flask.

### Conservative inoculum options

#### Standard option
- remove **25 mL** from the ~100 mL culture
- inoculate into **225 mL fresh F/2**
- final volume = **250 mL**
- refill the original backup culture with **25 mL fresh F/2**

#### Safer option for weaker strains
- remove **50 mL** from the ~100 mL culture
- inoculate into **200 mL fresh F/2**
- final volume = **250 mL**
- refill the original backup culture with **50 mL fresh F/2**

## Suggested strain-specific inoculum
- **SSB01, SSE01**: use the **25 mL into 225 mL** option
- **SSA01, SSA03**: use **25–50 mL into fresh F/2 to 250 mL**
- **SSA02**: use the **50 mL into 200 mL** option

## Transfer setup
Use the **Lab Companion BC-11B** 

## Materials
- Fresh sterile F/2 
- labeled **250 mL wide-mouth Erlenmeyers**
- sterile serological pipettes
- pipette gun
- marker / labels
- notebook / spreadsheet for records

## Labeling
Label each new flask with:
- strain
- medium
- date
- final volume
- initials
- “scale-up”

Label each original flask as:
- strain
- medium
- date
- “backup”

## Transfer steps
1. Prepare fresh sterile F/2.
2. Pre-label all new scale-up flasks and original backup flasks.
3. Work inside the **BC-11B**.
4. Gently resuspend the source culture.
5. Remove the chosen inoculum volume from the original ~100 mL culture.
6. Transfer it into the new scale-up flask.
7. Add fresh F/2 to the new flask to the target final volume (**250 mL**).
8. Refill the original flask with the same volume of fresh F/2 that was removed.
9. Swirl both flasks gently to mix.
10. Return both the new scale-up culture and the refilled backup culture to the incubator.
11. Record all volumes and notes.
